# 交互式关系图谱（D3.js版）
一款基于 D3.js v7 开发的可视化关系图谱，支持拖拽、缩放、悬浮提示等交互，适用于展示用户社交关系、游戏好友亲密度等场景。

## 功能特性
- 🎨 渐变背景+精致UI，视觉效果友好
- 🖱️ 节点拖拽、画布缩放，交互体验流畅
- 💬 鼠标悬浮or直接点击显示详细信息（关系、等级、亲密度等）
- 🎯 自定义关系类型、颜色、节点间距
- 📱 自适应屏幕尺寸

## 快速使用
1. 克隆/下载代码到本地
2. 直接打开 `蛋仔派对力导向图关系图谱.html` 即可运行（无需后端，纯前端）
3. 修改代码中「配置区」的内容，自定义：
   - 中心用户信息（昵称、头像、等级等）
   - 关系类型名称、颜色
   - 各关系下的好友列表

## 配置说明
| 配置项 | 说明 |
|--------|------|
| centerUser | 中心用户基础信息（昵称、头像、等级、性别、称号） |
| relationTypeMap | 关系类型映射（名称、颜色） |
| friendRelations | 各关系下的好友列表（id、昵称、头像、等级、性别、亲密度） |

## 技术栈
- D3.js v7（力导向图核心）
- HTML5 + CSS3（布局与样式）
- Vanilla JS（原生JavaScript交互）
